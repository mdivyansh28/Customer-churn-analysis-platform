# Customer Churn Analytics and Retention Intelligence Platform

## Project Overview

Customer Churn Analytics and Retention Intelligence Platform is an industry-level Data Analytics and Machine Learning project designed to help telecom organizations identify customer churn patterns, measure business performance, predict customer risk, and generate retention strategies.

This project is suitable for:

- MCA Final Year Major Project
- Data Analyst Portfolio
- Business Analyst Portfolio
- GitHub Portfolio
- Placement Interviews

---

## Objectives

- Analyze customer behavior
- Measure churn trends
- Track revenue impact
- Predict customer churn
- Generate retention recommendations
- Support business decision-making

## Installation

### Clone Repository

```bash
git clone https://github.com/username/customer-churn-project.git
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
